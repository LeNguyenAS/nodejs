## App structure
backend/
frontend/

## Install & Run the App
### install modules
`cd backend`
`sudo npm install`
### run server backend
`bin/www`

### install modules 
`cd frontend`
`sudo npm install`
`bower install`

### run server frontend
`grunt server`

### run generator document using jsdoc
`cd backend`
`./jsdoc.sh`

### Run server with mode test
`cd backend`
`./server-test.sh`

### Run server with mode development
`./server-dev.sh`
